## 설치

패키지 매니저를 통해 `@topinerd/react` 패키지를 프로젝트에 추가하세요:

```
yarn add @topinerd/react
```

그 다음, 패키지에서 제공하는 index.css 파일을 import 하세요:

```ts
import "@topinerd/react/index.css";
```

본 프로젝트는 [@vanilla-extract/css](https://vanilla-extract.style/) 라이브러리를 사용하여 스타일을 관리합니다. 이 라이브러리는 빌드 시 CSS 파일을 생성하므로, 각 컴포넌트에 정확한 스타일이 적용되려면 반드시 이 CSS 파일을 import 해야 합니다. 어플리케이션의 root 경로에 import 하는 방식을 권장합니다.

```tsx
import "@topinerd/react/index.css";

import App from "./app";

function main() {
  return (
    <>
      <App />
    </>
  );
}
```

이후, 패키지에서 컴포넌트를 가져와 필요한 곳에서 사용합니다.

```tsx
import { Button } from "@topinred/react";

const Hello = () => <Button>Hello World</Button>;
```
